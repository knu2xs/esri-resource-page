# ArcGIS User Types: Quick Reference Guide

## Overview

ArcGIS user types determine access levels and software entitlements for organization members. Each type includes specific capabilities and application access.

## User Types

### Viewer

**Purpose**: View and interact with maps and apps  
**Products Included**:

- ArcGIS Online web apps
- ArcGIS Field Maps (view only)
- ArcGIS Dashboards
- ArcGIS StoryMaps (view only)

### Creator

**Purpose**: Create, edit, and share maps and spatial content  
**Products Included**:

- All Viewer products
- ArcGIS Pro (desktop)
- ArcGIS Online Map Viewer
- ArcGIS Field Maps (full editing)
- ArcGIS Survey123
- ArcGIS QuickCapture
- ArcGIS Dashboards (author)
- ArcGIS StoryMaps (author)
- ArcGIS Notebooks
- ArcGIS Insights

### GIS Professional

**Purpose**: Advanced GIS analysis and workflows  
**Products Included**:

- All Creator products
- ArcGIS Pro Advanced
- ArcGIS Image for ArcGIS Online
- Additional extension licenses (Spatial Analyst, 3D Analyst, Network Analyst, Geostatistical Analyst)

### Insights Analyst

**Purpose**: Data-driven analytics and visualization  
**Products Included**:

- ArcGIS Insights
- Basic map viewing capabilities

## Choosing the Right Type

- **Viewer**: Consuming existing content
- **Creator**: Building maps and field data collection
- **GIS Professional**: Complex analysis and modeling
- **Insights Analyst**: Business intelligence and analytics

## Notes

!!! tip
    User types can be changed at any time by organization administrators.

!!! note
    Some products require additional licensing beyond user type entitlements.
