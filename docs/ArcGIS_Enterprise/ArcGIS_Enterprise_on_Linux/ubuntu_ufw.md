# UFW: Ubuntu Uncomplicated Firewall

References:

- [Ubuntu Server Documentation: Firewall](https://documentation.ubuntu.com/server/how-to/security/firewalls/)
- [Ubuntu Manpage: ufw](https://manpages.ubuntu.com/manpages/focal/man8/ufw.8.html)

On Ubuntu firewall management is simplified by using `ufw`, a user friendly way to manage the rules for `iptables`. This means, under the hood, `iptables` is still doing the filtering, but `ufw` is makes management much easier.

[application integration](https://manpages.ubuntu.com/manpages/focal/man8/ufw.8.html#application%20integration)
